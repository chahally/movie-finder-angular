## Movie Finder (Angular Project)

A modern Angular application that lets users search for movies using the TMDB API.

## Features

- Search for movies by title
- Display movie poster, rating, and release date
- Loading state while fetching results
- Error handling for API issues
- Responsive movie grid
- Clean, modular folder structure using standalone components

## Tech Stack

- Angular 21 (Standalone Components)
- TypeScript
- TMDB API
- RxJS for reactive programming
- HTML & CSS for styling

## Run Locally
1. Clone the repo
2. Install dependencies: npm install
3. Start the Angular development server: ng serve
4. Then open: http://localhost:4200
